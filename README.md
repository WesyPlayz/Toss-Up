# Toss-up
A Windows Forms Application in which you click and hold down on a button to charge it up, then release to toss it!

## Features :
Power - Holding down the button that reads "Click Me" will begin charging up the button's power, which it will begin to display on itself.
Toss - Once released the button, the button will be tossed across the window.
Score - Once the button lands, it will produce a score in which if is greater than ur current best score, will become your new best.
Intermission - While score is being displayed, the app will take a few seconds to show the current score and then reset the button.

## Prerequistits :
This app requires .Net 9.0 or newer and Windows 10 OS or newer.
